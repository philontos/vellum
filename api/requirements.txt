# --- At-rest encryption (optional) ---------------------------------------
# SQLCipher driver. Encryption is opt-in (set VELLUM_DB_KEY); without this
# package the app runs unencrypted, so it is intentionally NOT auto-installed
# here (keeps `pip install -r requirements.txt` and CI buildable everywhere).
# Easiest: run `./setup.sh`, which installs it (with the right PKG_CONFIG_PATH)
# and turns on encryption for you. To do it by hand instead:
#   macOS:  brew install sqlcipher pkg-config
#           PKG_CONFIG_PATH="$(brew --prefix sqlcipher)/lib/pkgconfig" \
#               pip install sqlcipher3==0.6.2
#   Linux:  apt-get install libsqlcipher-dev pkg-config && pip install sqlcipher3==0.6.2
